Dados

Dataset tratado, pronto para download e reuso — mesmos dados que alimentam este painel.

Dataset principal

violencia_rs_municipio_ano.csv — um registro por município × ano × tipo de crime, com casos, população e taxa por 100 mil hab.

⬇ Baixar CSV · Dicionário de dados completo

Colunas de violencia_rs_municipio_ano.csv
ColunaDescrição
municipioNome do município (maiúsculas, sem acento), ou NÃO INFORMADO
anoAno de referência (2012–2026)
tipo_crimeAmeaça, Estupro, Feminicídio Consumado, Feminicídio Tentado ou Lesão Corporal
casos_totalTotal de casos no ano, no município, para o tipo de crime
populacaoPopulação total do município no ano (estimativa IBGE)
taxa_por_100mil_habcasos_total / populacao × 100.000

Dados brutos e pipeline completo

As planilhas originais da SIP/PROCERGS e o código completo de extração/análise estão no repositório GitHub — inclui as tabelas de população (IBGE), a malha geográfica dos municípios, e todas as tabelas intermediárias em outputs/tables/ (regeneráveis rodando o pipeline localmente, ver README.md).

Ver repositório completo →

Antes de usar estes dados, leia as limitações

Taxa usa população total (não só mulheres); população de 2023 é interpolada; 2026 é ano parcial; "NÃO INFORMADO" não é um município real. Detalhes completos na página Metodologia.
