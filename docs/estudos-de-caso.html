Estudos de Caso

Recortes de um único município sobre a mesma metodologia (Camadas 1-3) aplicada ao estado inteiro — para checar se os padrões estaduais se replicam num município específico, ou não.

Esta página reúne estudos de caso municipais, todos derivados da mesma metodologia aplicada ao Estado do Rio Grande do Sul (ver Metodologia) — cada estudo filtra as mesmas tabelas/funções da pipeline para um único município e compara contra o estado, sem reimplementar nenhuma estatística. Nesta primeira versão, um único município: Uruguaiana. A estrutura foi pensada para receber outros municípios no futuro, seguindo o mesmo padrão.
